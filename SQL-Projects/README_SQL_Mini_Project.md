## Overview
This SQL mini project demonstrates core Data Analyst skills: aggregate functions, GROUP BY, HAVING, ORDER BY and using TOP for SQL Server. The dataset is a small orders table used to compute revenue and product/customer summaries.

## Dataset (orders)
Columns: order_id, customer, product, quantity, price

## Tasks / Queries
1. Total revenue
2. Revenue by customer
3. Top product by sales amount
4. Customers spending > 50,000
5. Product quantity summary

(See `orders_sales_summary.sql` for full SQL scripts.)
